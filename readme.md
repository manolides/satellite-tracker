# Satellite Tracker

A real-time 3D satellite tracking application visualizing the Airbus Pleiades Neo constellation.

## Features
-   **3D Globe Visualization**: Built with CesiumJS.
-   **Real-time Tracking**: Calculates satellite positions using TLE data.
-   **Realistic Rendering**: High-quality atmosphere, lighting, and night mode.
-   **Interactive**: Click on satellites to see their path and coverage.

## Setup
1.  Open `index.html` in your browser.
    -   OR run a local server: `python3 -m http.server 8080`
2.  Navigate to `http://localhost:8080`.

## Technologies
-   HTML5, CSS3, JavaScript
-   [Google Maps JavaScript API](https://developers.google.com/maps/documentation/javascript/overview)
-   [Satellite.js](https://github.com/shashwatak/satellite-js)
